Tesla Invader 3

by TeslaCigs

Commander (44,90€)


La Tesla Invader 3
est une box semi régulée à tension variable par Tesla.
Équipée d'un potentiomètre de 3.3 à 6.6 volts et compatible avec les résistances à partir de 0.1 ohm, la Tesla Invader 3 est capable d'atteindre jusqu'à 240W de puissance. Son switch, accompagné d'un chipset électronique assure toutes les protections nécessaires. La box Tesla Invader 3 est livrée sans accus.

BOX SIMPLE ET SÉCURISÉE

Malgré que ça soit une box semi régulée, la Tesla Invader 3 offre toutes les protections nécessaires pour une vape en toute tranquillité. La box possède les protections suivantes : inversion de polarité court-circuit surchauffe limite de courant limite de 10 secondes décharge importante d'accus

CONTENU DE LA BOITE

La box Tesla Invader 3 est livrée avec : 1 x box Tesla Invader 3 240W 1 x notice d'utilisation 1 x certificat d'authenticité

TECHNIQUES

La box Tesla Invader 3 est conçue pour fonctionner avec 2 accus en série. A l'arrière se trouve un capot aimanté qui cache l'emplacement doté d'une languette pour faciliter l'extraction. Tesla Invader 3 n'est pas dotée de chargeur, il est nécessaire d'utiliser un chargeur d accus externe. Il est recommandé d'utiliser des accus avec au moins 30A de décharge. La box Tesla Invader 3 est livrée dans un boitier en alliage de zinc avec une très belle finition. Simple mais très solide, la box affiche un poids de 275g avec les accus (non livrés). Les angles arrondis du boitier offrent une bonne prise en main.
